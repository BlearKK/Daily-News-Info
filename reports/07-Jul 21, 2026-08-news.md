<h3 id="blog">精选博客 Blog</h3>
<h4 id="aiml">AI/ML 与 边缘计算</h4>
<p><strong><a href="https://blogs.nvidia.com/blog/siggraph-news-2026/">NVIDIA 在 SIGGRAPH 推出 Cosmos 3 Edge，以 Agent 化与具身智能革新图形与模拟技术</a></strong>
在今年的 SIGGRAPH 大会上，NVIDIA 宣布了一系列重大 AI 进展，重点展示了 Model Context Protocol (MCP) 协议在主流创意软件中的落地，以及新一代边缘世界模型 Cosmos 3 Edge，标志着 AI 正在从“加速工具”彻底演变为“协作 Agent”。</p>
<ul>
<li><strong>MCP 创意生态成型</strong>：Adobe、Canva、Blender、Unreal Engine 等全面接入 MCP 协议，AI 助理能直接在 3D 资产、动画管线中执行审查、清理等复杂多步任务。</li>
<li><strong>Cosmos 3 Edge 边缘世界模型</strong>：发布 40 亿参数的多模态世界模型，在 VANTAGE-Bench 视觉分析评测中位列同尺寸第一，可在 Jetson Thor 等边缘设备上实现超高吞吐量的实时机器人控制。</li>
<li><strong>本地 Agent 全栈落座</strong>：推出基于 DGX Station 的 NemoClaw 与 Nemotron 3 Ultra (550B) 本地 Agent 构建方案，实现 100% 本地运行、无需联网。</li>
<li><strong>标签</strong>：#AgenticAI #PhysicalAI #SIGGRAPH #Cosmos3</li>
<li><strong>来源与时间</strong>：NVIDIA Blogs · 3 小时前</li>
<li><strong>延伸来源</strong>：<a href="https://huggingface.co/blog/nvidia/cosmos3edge">Introducing Cosmos 3 Edge - Hugging Face Blog</a></li>
</ul>
<h4 id="-1">生命科学与算力基建</h4>
<p><strong><a href="https://blogs.nvidia.com/blog/bristol-myers-squibb-building-life-science-industrys-most-advanced-ai-factory-on-nvidia-vera-rubin/">百时美施贵宝（BMS）正基于 NVIDIA Vera Rubin 架构构建生命科学界最先进的 AI 工厂</a></strong>
制药巨头 BMS 宣布部署其第二套 NVIDIA DGX SuperPOD 系统，该系统由 8 台 DGX Vera Rubin NVL72 系统组成。该平台将全面向公司内部所有科学家开放，消除了算力等待瓶颈。</p>
<ul>
<li><strong>超强能效比</strong>：基于 Vera CPU 和 Rubin GPU 的系统，每兆瓦性能较之前架构提升了高达 10 倍。</li>
<li><strong>“预测优先”与多模态加速</strong>：BMS 将利用 AI 大幅扩展抗癌 CELMoD 化合物库。通过 AI 代理跨越数据孤岛，将原本需要数周的靶点识别工作压缩至几天。</li>
<li><strong>标签</strong>：#VeraRubin #Supercomputing #LifeSciences #DrugDiscovery</li>
<li><strong>来源与时间</strong>：Brian Caulfield · 8 小时前</li>
</ul>

<h3 id="news">今日要闻 News</h3>
<p><strong>1. <a href="https://techcrunch.com/2026/07/20/google-is-working-on-a-new-ai-chip-designed-to-make-gemini-more-efficient/">谷歌秘密研发“Frozen v2”AI 芯片，拟将 Gemini 架构直接写入硅片以极度优化能效</a></strong>
谷歌正在秘密开发代号为“Frozen v2”的新一代服务器芯片。该芯片尝试将 Gemini 的神经网络架构直接硬编码到硬件硅片中，旨在将推理能效提升 6 到 10 倍。</p>
<ul>
<li><strong>大幅削减推理成本</strong>：该芯片预计于 2028 年问世，若成功将使谷歌在与 OpenAI、Anthropic 的 AI 价格战中获得决定性的成本优势。</li>
<li><strong>特定硬件绑定</strong>：硬编码架构虽然带来极致效率，但也意味着一旦 Gemini 架构发生重大演进，芯片可能需要重新设计。</li>
<li><strong>标签</strong>：#GoogleGemini #Silicon #TPU #AIChips</li>
<li><strong>来源与时间</strong>：TechCrunch / The Decoder（合并报道）· 2 小时前</li>

</ul>
<p><strong>2. <a href="https://techcrunch.com/2026/07/20/hugging-face-confirms-breach-affected-internal-datasets-and-credentials-urges-users-to-take-action/">Hugging Face 遭受自主 AI Agent 漏洞攻击，并首次使用 AI 防御系统反击成功</a></strong>

<ul>
<li><strong>全自主 AI 攻击</strong>：黑客部署的 Agent 框架自动发起了数千次协同漏洞扫描与利用。</li>

<p><strong>3. <a href="https://www.technologyreview.com/2026/07/20/1140655/ai-biases-hiring-humans/">普林斯顿与芝大最新研究：o3、R1 等高推理 AI 模型在招聘决策中比人类更容易形成严重偏见</a></strong>

<li><strong>强化目标优于灌输价值观</strong>：仅在 Prompt 中警告模型“要公平”几乎没有效果，但为模型设定“多元化招聘可获得额外奖励（Reward）”的目标能大幅减轻偏见。</li>
